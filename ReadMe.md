该项目介绍如何使用ReadtheDoc中的sphinx来编写技术文档

同步命令：
1. 查看当前`branch`
```
git branch
```
带`*`的表示为当前工作分支

2. 将当前`branch`推送到远程仓库，同步`master`分支
```
git push --set-upstream origin mybranch:master
```
